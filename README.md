# MY Online Store

Yeh mera online store ka web project hai. Isme maine mobile phones aur premium charging adapters sell karne ke liye ek modern website banayi hai.

## Project Features
* **Glassmorphic UI:** Website ko ek premium look dene ke liye cards par glass effect lagaya hai.
* **Fully Responsive Menu:** Mobile screens ke liye ek clean responsive navbar set kiya hai.
* **Multi-Page Layout:** Isme main page ke sath-sath About Us, Feedback, Customer Service, aur Contact Us ke pages bhi inline styling ke sath jode hain.

## Tech Stack
* HTML5
* CSS3 (Flexbox Layout & Inline CSS)
