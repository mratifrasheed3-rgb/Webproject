# MY Online Store

This is my e-commerce web project. I have created a modern online store website to showcase and sell mobile phones along with premium charging adapters.

## Project Features
* **Glassmorphic UI:** Implemented a sleek glass-effect on cards to give the website a premium and professional look.
* **Fully Responsive Menu:** Configured a clean, responsive navbar optimized perfectly for mobile screens.
* **Multi-Page Layout:** Connected the main landing page with dedicated sub-pages including About Us, Feedback, Customer Service, and Contact Us, all styled beautifully using inline CSS.

## Tech Stack
* HTML5
* CSS3 (Flexbox Layout & Inline CSS)
